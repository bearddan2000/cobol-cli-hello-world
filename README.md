# cobol-cli-hello-world

## Description
This is a POC project to demonstrate gnu cobol.

## Tech stack
- cobol
- cobc

## Docker stack
- ubuntu

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
